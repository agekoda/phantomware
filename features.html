Features

A deep dive into Phantomware's capabilities and specifications.

Core Features

ESP32-S3 Microcontroller

Powerful dual-core Xtensa LX7 processor with integrated Wi-Fi 4 (802.11n) and Bluetooth Low Energy 5 (BLE) for high-performance computing and connectivity.

Robust Power System

Features a substantial 3000mAh LiPo battery with modern USB-C charging (TP4056 charger) and a precise MAX17055 fuel gauge for accurate battery monitoring. Includes essential overcharge and overcurrent protection for enhanced safety and longevity.

Manual Power Switch

A dedicated hardware switch provides true power shutdown, ensuring complete control and zero idle consumption when off.

Inputs & Controls

Navigation Buttons

Equipped with 6 tactile push buttons for intuitive navigation and precise control over the device's interface and functions.

Infrared (IR) Transceiver

Integrated with a TSOP75438 receiver and 3 high-power VSMY14940 LEDs, enabling versatile capabilities for both receiving and transmitting IR signals, perfect for universal remote control and detailed IR data analysis.

NFC Reader/Writer

Incorporates the PN532 NFC chip via an SPI interface, allowing for efficient reading, writing, and emulation of various NFC tags and cards.

Temperature Sensor

Includes an MCP9701 temperature sensor for accurate monitoring of ambient or board temperatures, crucial for environmental data logging and system health. -10℃ - +125℃.

Visual & Audio Feedback

High-Resolution Display

A vibrant 240x320 TFT display (ST7789) provides a crisp, colorful screen for rich user interfaces, detailed data visualization, and interactive applications. Max SPI speed is ~ 60 MHz.

Dual Status LEDs

Features dedicated green and red LEDs for immediate visual feedback, indicating system states, activity, and notifications.

Piezo Buzzer

A versatile piezo buzzer offers audible feedback for alarms, button presses, system alerts, and even custom sound effects or simple tunes. 100-4000 Hz, 75 dB.

Storage & Connectivity

Micro SD Card Slot

Provides expandable storage for extensive data logging, loading custom firmware, managing configuration files, and handling large datasets. Supported file systems include FAT12, FAT16 and FAT32.

SPI

The full SPI bus is efficiently shared between the PN532 NFC and the SD card module, utilizing dedicated Chip Select (CS) pins for seamless operation of both peripherals. These can be controlled in software.

USB-C Port

A USB-C port for reliable power delivery, fast data transfer, and serial communication with development environments.

Expandability & Debugging

GPIO Expansion Pins

Accessible General Purpose Input/Output (GPIO) pins are broken out for connecting external modules, sensors, custom circuitry, and for signal analysis, extending Phantomware's capabilities.

Communication Breakouts

Dedicated I²C, SPI, and UART breakout pins facilitate easy integration with a wide range of external peripherals and communication protocols.

Maximized GPIO Utilization

All relevant GPIOs are assigned and exposed to ensure maximum functionality and ease of use for developers building complex projects.
