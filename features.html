Phantomware Features

A deep dive into Phantomware's capabilities and specifications.

Core Features

  • ESP32-S3 microcontroller with Wi-Fi 4 & BLE 5
  • 3000mAh LiPo with USB-C charging & 3.3V regulator
  • Manual power switch for true shutdown

Inputs & Controls

  • 6 tactile push buttons for navigation and control
  • IR receiver (TSOP75438) & 3× high-power IR LEDs for remote control
  • PN532 NFC reader/writer over SPI
  • Temperature sensor MCP9701 for ambient/board temperature

Visual & Audio Feedback

  • 240×320 TFT display for rich UI
  • Dual status LEDs (green & red) for system state
  • Piezo buzzer for alarms, feedback, and tunes

Storage & Connectivity

  • Micro SD card slot for data logging and file storage
  • Full SPI bus sharing PN532 & SD card with dedicated CS pins
  • USB-C port with CC line detection for charging and data

Battery & Power Management

  • MAX17055 fuel gauge for precise battery monitoring
  • Built-in charging current configuration and status LEDs
  • Overcurrent and thermal protection

Expandability & Debugging

  • GPIO expansion pins for external modules
  • I²C, SPI, UART breakout pins for external modules
  • All GPIOs assigned for maximum functionality
